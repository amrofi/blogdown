Introdução

Prezados leitores. Iniciei essa jornada de fazer um website com o pacote blogdown e estou aos poucos migrando minhas informações para o site. Fiz várias postagens utilizando a ferramenta de publicação com o RStudio e o RMarkdown colocando no Rpubs: http://rpubs.com/amrofi/. Agora estou aos poucos elaborando este site para melhor apresentar meu trabalho a vocês. Espero que tenham paciência pois ainda estou aprendendo a mexer no site. Inicio pelo código de religião com text mining pois sem Deus nada disso seria possível.
